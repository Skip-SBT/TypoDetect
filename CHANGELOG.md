## [1.0.0] - 2024-01-01

### Added
- Initial release of TypoDetect.
- Real-time typo detection for a configurable target word using fuzzy matching.
- Red screen-border flash on typo detection.
- System tray icon (green = monitoring on, grey = monitoring off).
- Adjustable strictness: Strict, Normal, Relaxed.
- Persistent settings saved across restarts.
- Built-in activity log with quick open/clear actions from the tray menu.
- Crash logging and heartbeat monitoring.
- Auto-recovery watchdog for input hooks.
- Windows auto-start support via the startup folder.
